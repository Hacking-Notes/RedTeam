<h2>Type of API's</h2>

RESTful API
- Most common API
- Uses JSON
- Has a defined structure in request

GraphQL
- Newcomer on the scene
- Uses a custom query language
- Single endpoint powers all of the API

Others
- Standalone JSON calls
- Websocket are also used as API's
